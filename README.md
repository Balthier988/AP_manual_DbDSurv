# AP_manual_DbDSurv
This is the manual apworld of Dead by Daylight Survivor for Archipelago, both in english and in french

# Archipelago Global Setup
This guide assumes you have experience with both Archipelago and Manual for Archipelago. If you have no experience with either of these, follow the Archipelago setup guide found here: https://archipelago.gg/tutorial/Archipelago/setup/en

Or here for Mac users: https://archipelago.gg/tutorial/Archipelago/mac/en

After completing the Archipelago setup guide, follow the Manual for Archipelago setup guide found in the Manual for Archipelago Discord Server: https://discord.gg/T5bcsVHByx

You will also need the Universal Tracker for Archipelago: https://discord.com/channels/731205301247803413/1367270230635839539

# Global Dead by Daylight Info
Items : Every Perks (Base Perks / DLC Perks included), Every Survivors, Progressive Items (Map / Key / Fog Vial / Toolbox / Medikit / Flashlight), Progressive Add-ons, Progressive Offerings

Starting items : You'll start with one survivor and three perks

Locations : Win (Escape) with every Perks, Win (Escape) with every Survivors, 27 objectives to do during a trial

Traps : There is no traps YET

Goal : Currently need to win with every survivors up to Rick & Michonne Grimes (The requirement is only to have those but you still have to win with them)

It can be played solo or coop, but beware, it's harder to escape when you are 4 Survivors with 4 "random" perks, no items, no add-ons and no offerings

There is currently no settings, because i'm not English and i struggle really hard to make those, if someone have the  knowledge to do it, it'll be delightful to shatter it myself to learn



# Informations Dead by Daylight #FR

Items : Chaque Compétences (Perks de Base / Perks de DLC inclus), Chaque Survivants, Objets progressifs (Carte / Clé / Fumigènes / Caisse à Outil / Medikit / Lampe de poche), Add-ons Progressifs, Offrandes Progressives

Items de départ : Vous commencerez avec un survivant ansi que trois compétences

Checks à envoyer : S'échapper avec chaque compétences, S'échapper avec chaque survivants, 27 objectifs à faire durant les parties

Pièges : Pas de pièges pour l'instant

But : Il est nécessaire de gagner avec chaque Survivants, comptant les derniers sortis : Rick & Michonne Grimes (Vous aurez la capacité de valider le Goal une fois tous les Survivants débloqués, mais gagnez d'abord avec tous !)

Il peut être joué en solo ou en coopération, cependant, faites attention, il est plus dure de s'échapper quand vous êtes 4 survivants avec 4 perks "aléatoires", aucun objet / add-ons / offrandes
